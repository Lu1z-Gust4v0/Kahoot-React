# Kahoot React Clone

## Description 

This a kahoot clone made in NextJs with a Go backend. Realtime updates are handled using websockets

## Technologies 

- NextJs
- TailwindCSS
- Go 
- Fiber
- Postgresql
- Docker

## Running 

- Create a .env where .env.example exists

- Run Postgresql and go docker containers
```
cd backend/
docker compose up
```

- Run frontend
```
npm run dev
```
